[딥러닝 코드 개발 프롬프트]

[1] horse_or_human  : 말과 사람 이미지 분류 모델
    프로젝트 이름 :  Horse_Or_Human_Project
    아래 URL에서 데이터를 다운받고 압축해제 후 프로젝트폴더에 복사해 놓는다  
    https://www.kaggle.com/datasets/anhkhoango/horse-or-human?select=horses

(프롬프트)
horses_or_humans 폴더안에 있는 이미지를 사용해서
말과 사람을 구별하는 딥러닝 분류 모델을 만들어줘
구글 Tensorflow를 사용해서 소스를 만들어줘.
모델을 평가하고 평가 결과를 출력하는 코드를 구현해줘.

필요한 파이썬 모듈을 requirements.txt에 넣어 저장해줘

이소스를 기반으로 웹페이지에서 말이나 사람 사진을 업로드하면
분류하는 앱을 따로 만들어줘. Streamlit을 사용해줘

- 오류 수정: 정답이 사람인데 말만 나와. 수정해줘

[2] cats_and_dogs_filtered  : 말과 사람 이미지 분류 모델
    프로젝트 이름 :  Cats_And_Dogs_Project
    아래 URL에서 데이터를 다운받고 압축해제 후 프로젝트폴더에 복사해 놓는다  
    https://www.kaggle.com/datasets/birajsth/cats-and-dogs-filtered

(프롬프트)
-1단계:
cats_and_dogs_filtered 폴더안에 있는 이미지를 사용해서
고양이와 강아지를 구별하는 딥러닝 분류 모델을 만들어줘
구글 Tensorflow를 사용해서 소스를 만들어줘.
모델을 평가하고 평가 결과를 출력하는 코드를 구현해줘.

필요한 파이썬 모듈을 requirements.txt에 넣어 저장해줘

-2단계:
정확도가 0.72로 너무 낮아. 정확도를 0.9 이상으로 높이도록 수정해줘
구글 인셉션 모델을 사용하여 전이학습 모델로 다른 파일로 다시 구현해줘.

- 오류슈정 :정확도가 너무 낮아 0.52야 다시 수정해줘

-3단계: (실습 생략)
이소스를 기반으로 웹페이지에서 고양이나 강아지 사진을 업로드하면
분류하는 앱을 따로 만들어줘. Streamlit을 사용해줘
